# WebGL Tunnel Rush
A game made in WebGL with lighting, flashing similar to tunnel rush.

## Controls
- a - move left
- d - move right
- w - jump
- g - change display to grayscale
- t - toggle texture

## About the game
- Score as much as possible.
- Travelling speed increases with time.
- Multiple levels
- Multiple obstacles
- Texture and lighting on tunnel.
- Periodic flashing.
- Grayscale toggling.
